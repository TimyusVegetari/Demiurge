Code Style Guide for Demiurge

Header Files

C++ header files have the extension .hpp and are structured as follows:

  1. License block
  2. Doxygen minimale class presentation
  3. Opening include guard
  4. Included headers
  5. Opening namespace gm
  6. One or more of the following:
    • Class and type definitions
    • Global function declarations
    • Nested namespaces
  7. Closing namespace gm
  8. Closing include guard
